# 🐈 CatTranslator : 고양이 번역기
> 고양이의 울음소리를 녹음해 감정을 분석하고 번역해주는 앱입니다.
울음은 주파수 기반으로 분석되며, 그 결과는 GPT를 통해 해석되어 감정으로 표현됩니다.
분석된 감정은 텍스트와 그래프로 시각화되어 사용자에게 직관적으로 제공됩니다.
